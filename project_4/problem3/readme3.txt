This script pings the Google DNS server (8.8.8.8) ten times and returns the average ping time in whole milliseconds.
I may rewrite it using awk sometime to allow for floating point math, but this is a funcitonal proof of concepts.

I put a few concepts that I wanted to practice into use here including text manipulation (sed and awk), shell arithmetic, and looping.

I'm sure there is a more efficient way to do all that, but as a learning excercise it was really helpful.
